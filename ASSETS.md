# ASSETS

The license record for every file under `public/assets/`. See `GRAPHICS_SPEC.md §9.3` and `§13` for the policy and the sourcing plan.

## Policy

Every shipped asset is one of:

- **Original** — hand-authored or generated by us.
- **Commissioned** — made for us under a license we hold.
- **Permissively licensed** — CC0, or CC-BY with attribution, compatible with this repo.

Never Valve's or Blizzard's files: not models, textures, icons, fonts, or sounds. Heroes resemble their archetype through our own and CC0 assets, never extracted files (`§1.1`).

The build boots and looks intentional with `public/assets/` empty. Assets are an enhancement over the procedural base, never required (`§9.5`).

## Pipeline

Raw packs are downloaded into `tmp/asset_src/` (gitignored, never committed) and optimized into `public/assets/` by `scripts/assets/build_assets.mjs` (gltf-transform: resample, prune, dedup, meshopt; sharp: resize to webp). One spec JSON per pack lives under `scripts/assets/specs/`. Commit only the optimized output and the spec, plus a row in the table below. The build writes `public/assets/manifest.json` so runtime and review tools can see each shipped file, byte size, type, source spec, and preload group.

```sh
npm i -D @gltf-transform/core @gltf-transform/extensions @gltf-transform/functions meshoptimizer sharp
npm run build:assets -- scripts/assets/specs/<pack>.json
npm run assets:check
```

## Directory layout

```
public/assets/
  heroes/    hero GLBs           (matches assets.ts manifest: /assets/heroes/<id>.glb)
  creeps/    creature/creep GLBs
  props/     foliage, rocks, town buildings, set dressing
  env/       HDRI sky / IBL maps
  textures/  terrain PBR sets, water normals, shared maps
  ui/        fonts (woff2), non-generated UI textures
  vfx/       particle / telegraph atlases
```
